ctrl + k + u = Descomenta a linha

alt + seta para cima ou seta para baixo = Move a linha

alt + shift + seta para baixo = duplica a linha 

ctrl + b = Esconde o menu lateral

ctrl + shift + c = Abre o terminal externo

ctrl + f = Procura qualquer palavra
